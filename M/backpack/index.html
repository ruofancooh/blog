如此背包

发表于 2023-11-27 更新于 2023-12-02 分类于 音乐

《如此背包》

大伙晚上好，给大伙来说段相声

先做个自我介绍，你先吧，你是干什么工作的

（我啊，我是学生，学计算机的）

学生，怎么上这儿来说相声了

（大学嘛，不是什么都要学嘛。学说相声，学写相声，也算是学习。）

啊，有理

（那你是干什么工作的）

我也是学生，哎我们不是一个人吗

（啊，这我还没注意）

这个说相声啊，讲究什么，你知道吗

（说学逗唱啊）

说相声，讲究抖包袱

（抖包袱，包袱是什么意思啊？）

包袱，就是笑点哪，就是让观众朋友们图一乐

（哦，说相声就是为人图一乐，那你这不是在解释笑话吗）

啊这个……在算法里啊，还有一个专业名词（什么名词啊）背包问题

（背包问题？什么意思）

就是有一个背包，承重有限。还有几个不同重量的物品，这些物品的价格也是不同的

（您等会，能具体一点吗？）

具体一点

（就是举一个例子）

你，是一个小偷，把人家屋子的锁给撬了，顺便把保险柜也撬了。

（什么保险柜这是）

打开保险柜，八十根金条，五十个大钻，二十张银行卡，映入眼帘。

但是，你这一趟来得比较仓促，什么口袋都没拿，浑身上下只有两个衣服口袋能装东西。

（我怎么还不提前准备呢）

刚偷完上一家，口袋给同伙拿了，再寻思赚点外快。

（嚯）

现在你得考虑了，怎么往口袋里装这些东西，使得在法网网到你之前你能够最大程度的挥霍。

（那我可挥霍不了几天了）

（那这么说，背包问题就是小偷问题）

对喽，你可以这么记。把背包问题挪到说相声上，也是同一个理。

（什么理）

你想想，背包，就是包袱。

（包袱）

包袱里的东西，是什么？是嘴里头说的，是语言。

所以相声上的背包问题，就是如何使用最简单的语言，抖出一个最响的包袱。

（啊这……有点……道理）

相声是什么？是语言的艺术。

（噢，是说的道理）

背包问题，背包，是先把包袱背上了才能抖下来。

（那是怎么背上包袱的呢）

看代码：（代码？）

dp[i][j]

（dp[i][j]？dp 是什么意思）

dynamic programming

（什么玩意儿？呆什么）

英语！不懂吧？就是动态规划的意思

（动态规划，什么叫动态规划啊）

规划，就是计划你之后要干什么事儿

（那动态呢）

动态

（噢，那我要是计划我今天晚上吃什么饭，是不是也叫动态规划）

对喽（啊）

这动态规划有一特点，就是要记住过去的事儿，然后通过过去的事儿来规划将来的事儿

（什么意思）

比如你用哔哩哔哩，点那个搜索框，是不是出来了一些词儿啊。

（都是我搜过的）

它是不是还给你推荐新词儿啊，这就是他们那边的动态规划。

（哦）

你这边的动态规划呢，就是你用浏览器，点那个历史记录，然后（哎哎哎）怎么了？

（这 ei 个不能点。）

这怎么不能点了呢？

（额，我开的是无痕模式。）

无痕模式，那你是怎么规划的呢。

（我是，直接输网址）

然后呢

（然后……）

发我一份

（发什么啊发你一份）

有福同享啊

（享什么啊你，继续讲背包问题啊）

好好，开不起玩笑，继续说这个背包问题。背包问题要用动态规划来解决，这个动态规划的英文缩写，是 dp

（就是那什么呆）

dp，不就是大盘吗

（大盘？怎么说）

有一特别火的射击游戏，叫和平精英，里面有一把枪，叫 dp-28 不是叫大盘鸡吗。

（哦，dp 可以谐音成大盘）

大盘，不得有点掌控全局的意思吗。（哦）只要拿到大盘鸡，这盘鸡就稳了。

（那只要使用动态规划，解决这个背包问题就稳了？）

有慧根

（还慧根）
