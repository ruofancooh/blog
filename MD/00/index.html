Windows 10 环境配置

  1. 下载 MongoDB，我下载的是7.0.0 (current)的zip格式。
  2. 解压，右键用 PowerShell 运行Install-Compass.ps1，安装 GUI。 默认会安装到 C 盘，安装完后手动移到 D 盘。
  3. 建数据库文件夹，建一个空日志test.log。
  4. 启动 mongod
  5. 用 MongoDBCompass 连接。

启动脚本

mo.bat

脚本会 call mongod，停止则断开连接。
