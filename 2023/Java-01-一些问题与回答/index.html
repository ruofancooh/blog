一些问题与回答

main函数不加static可以吗？

不行。程序启动时实例还没有被创建，需要通过类名调用静态方法。

当类名和文件名不一致时，代码检查器会提示修改。那么.java文件里只能写一个类吗？

只能写一个与文件名相同的公共类。其他的类不加public。
