所有心理问题都来源于：

MMPI

统计学家 George Box 说：所有模型都是错的，有些是有用的。

这里研究的是对于明尼苏达多相人格测验（Minnesota Multiphasic Personality Inventory）的一系列判断题，回答【是】或【否】对最终结果的影响。依据是知乎网友 aweeee分享的 xlsx 文件。

  1. 做一道判断题，回答是或否

  2. 答完一道题之后，对某一个或者某几个量表加 1 分或不加分

  3. 答完所有题之后，会得到十四个分，包括四个效度量表（Q L F K）和十个临床量表（Hs D Hy Pd Mf Pa Pt Sc Ma Si）。这十四个分称作【原始分】。

    • 如果你做的是 566 道题的，额外会有五个附加量表（MAS Dy Do Re Cn）。笔者做的都是两百道题的，只有这十四个分。

  4. 特别地，对这五个临床量表加上一定比例的 K 分，加完之后的称为【K 校正分】

    • Hs += 0.5K
    • Pd += 0.4K
    • Pt += 1.0K
    • Sc += 1.0K
    • Ma += 0.2K

  5. 现在我们有十四个分。对除 Q 之外的十三个分分别计算 T 分：

    \[ T = 50 + \frac{10 \times (你的得分 - 常模均值)}{常模标准差} \]

    其中，【常模均值】和【常模标准差】由你的国籍、性别和年龄范围选择。

    【常模均值】是一群正常人（健康人）做的平均分。用你的得分与其相减，刻画的是你与正常人的偏离程度。

    【常模标准差】刻画的是这一群正常人得分的离散程度。用其作分母，可以缩小误差。

四个效度量表，测你做的结果是否有效：

Q（Question）【疑问】：你没有回答的题目数

在实际测验中，与“性”有关的几个题目可以选择跳过回答。

L（Lie）【说谎】：你往圣人方向靠的程度

例如

做游戏的时候，我只愿赢而不愿输

偶尔我会想到一些坏得说不出口的事

有时我也讲假话

等的这一些问题，回答【否】会加分

F（Validity）【诈病】：你装病的程度，或者判断力丧失的程度

例如

有人一直想毒死我

有时我觉得有鬼神附在我身上

我能在我周围看到其他人所看不到的东西、动物和人

等的这一些问题，回答【是】会加分

K（Correction）【校正】：你对自己的私事隐瞒的程度、防御的程度

例如

当我正在做一件重要事情的时候，如果有人向我请教或打扰我，我会不耐烦的

我觉得许多人喜欢夸大自己的不幸，以便得到别人的同情和帮助

我不在乎别人对我有什么看法

回答【否】会加分

我几乎没有和家里人吵过嘴

回答【是】会加分

十个临床量表，测你病的严重程度：

Hs（Hypochondriasis）【疑病】：你对自己身体过度关心的程度

例如

我的胃口很好

我睡得不安，容易被惊醒

我的身体和我的大多数朋友一样地健康

我常觉得头上好象有一根绷得紧紧的带子

我不容易疲倦

有时候我觉得我的头顶一碰就疼

问的都是与你身体相关的，如果你给的反常回答太多了：

  • 如果你说的是实话：说明你确实有这些症状，说明你很反常
  • 如果你说的是中话：说明你在过度关心得病，说明你很反常
  • 如果你说的是假话：说明你在诉苦，说明你很反常

D（Depression）【抑郁】：你抑郁的程度

前 200 道题里，有 44 道题涉及对【抑郁】这一项加分。

  • 笔者有一次抑郁的得分是 22，最终【正常】的概率在 90% 以上
  • 另一次得分 38，【焦虑抑郁障碍】的概率在 90% 以上

说明这是一个参量。我们来关注几个同时对其他很多参量造成影响的题目：

我发现我很难把注意力集中到一件工作上

  • 如果回答【是】：【抑郁】【癔病】【精神病态】【精神衰弱】【社会内向】【外显性焦虑】加一分
  • 如果回答【否】：【支配性】加一分

我希望我能像别人那样快乐

  • 如果回答【是】：【抑郁】【精神病态】【精神衰弱】【社会内向】【外显性焦虑】【依赖性】【控制力】加一分

我经常是快乐的

  • 如果回答【否】：【抑郁】【癔病】【精神病态】【偏执】【外显性焦虑】【依赖性】加一分

许多时候，我觉得浑身无力

  • 如果回答【是】：【疑病】【抑郁】【癔病】【精神衰弱】【依赖性】加一分

Hy（Hysteria）【癔病】你“天真幼稚”、“歇斯底里”的程度

前 200 道题里，有 60 道题涉及对【癔病】这一项加分，得分 30 以上说明你有大异常。

其中答【是】加分的有 11 道，有 4 道同时对【疑病】加分，另外 7 道所描述的症状比【疑病】更反常、更“躯体化”、不属于健康人应有的，例如：

知识的诅咒

如果一个人从来没见过苹果，他听到苹果两个字的声音，他会不知道是什么。

如果一个人没学过藏语，他听到 ཀུ་ཤུ།的声音，他会不知道是什么。

你用的专业名词，别人可能不知道是什么意思。所以，讲课真的很难。

最好的就是【举一个例子】。所以，当你看到笔者的笔记里没有例子时，说明笔者压根就没弄明白。

就算有例子，也可能是望文生义，照一个名词套。哲学、心理学相关名词尤甚。比如【自我指涉】

用的带太多语气、太多连接感、太多承上启下感、不简洁的，通通说明不懂。

自我指涉

自我指涉是什么意思？

你可能会注意到，这个网页的名字叫 this.html

刘慈欣在《三体》第二部里写到，写小说的人是作为一个观察者，记录下他脑子里上演的东西。

类似地，笔者在这个环境里，营造的是一个虚假的自我、理想的自我。实际上笔者把时间都花在打字上了。为了打出这一个标题下面的字，可能会花费好几个小时。

进而，对于笔者打出的有些字，是通过前面的语境续写的，而不一定是笔者真实经历的。

有一首歌叫《不完备定理》，笔者是有类似的想法的，但是根本达不到那个境界——这是人家用实力写出来的歌，不是你写的。就像，你读了一本《名人传》，就能认为你是名人吗。

专辑的名字叫《文理叙术 The Contextual Order》，意思是【上下文顺序】。曲风属于【前卫摇滚】——在笔者看到的互联网语境下，这不是一个好名词。这跟吗啡差不多，在紧急状态之下可以镇痛，滥用就变成毒品了。

写小说，前提是你之前读过非常多的文学作品。像笔者这样，属于前言不搭后语。

写论文为什么要用【笔者/我们】，而不是【我】？

如果用后者：一是，会显得你很自私。二是，读你东西的人，会代入不了语境。

思维反刍的成因

思维反刍的意思是：一直重复地想同一件事

因为你想得到一个结果，但是一直没有得到。

比如：你在想你之前的不愉快经历。这是因为你想得到一个更好的结果，所以在脑子里一直重演。

又比如：你之前在互联网上发了一些东西，有很多人回复你。但是，【发出东西】这个行为是通过【想】+【动动手指】实现的，并没有经历【通过声带说话】这一步。

这就造成一种条件反射：只要想了，就有人回复你了

所以，当没有人回复你的时候，你就会一直想。

这和使用搜索引擎/点视频看差不多。打几个字/点进去，立马就得到了。当你想睡觉的时候，放下手机——麻了。

笔者父亲小时候也有类似的经历。随着之后的工作和生活，他的注意力被转移了。现在回想起来，这是因为他太孤单了。

关于边听歌边睡觉，直至入眠

这是因为，你的注意力集中在了音乐身上，你就不会想别的事。

但是——如果你听到一个量了，你可能会觉得，之前、未来听的一些歌没有意思了，就是吃菜吃腻了。你的注意力会继续发散。

你脑子里有一个缓冲区，你可以通过这个缓冲区，预测到歌曲下一步的发展。

当歌曲下一步的发展，与你预测的有一定区别时，你才会觉得歌曲有一种新鲜感。

对于睡觉，一直不新鲜、一直新鲜不一定是好事。因人而异

——这是一种极其错误的说法。就跟你读了几本小说，就认为世界上所有小说都没有意思了一样。

还是听得太少。

C 语言中的指针

普通变量

变量类型int，变量名num->变量值100。

普通变量名已经有一层指向了。

指针变量

用于专门存放内存地址（值的位置）的变量，叫指针。

变量类型int*，变量名ptr->变量值num->100的内存地址

（这种说法好像是不严谨的，int*好像是 C++ 的说法）

&num，就是获取num值的位置。

  • &一个变量名，就是获取变量名的值的地址。简称取地址。
  • *一个变量名，就是获取变量名的值（只对指针有效）。简称访问地址。
  • *&一个变量名就是先&后*（两个运算符优先级相同，且都是右结合）。
    • 先取地址，后访问地址，和单独使用变量名效果一样。
  • &一个指针变量名，就是获取指针变量值本身的位置（存放num地址值的地址）。

还可以创建二级指针，就是指向指针的指针：

  • pptr变量名指向某一个地址，这个地址处记录着ptr的地址
  • ptr变量名指向某一个地址，这个地址处记录着num的地址
  • num变量名指向某一个地址，这个地址处记录着num的值100

不要开辅助瞄准

辅助瞄准就是算法帮你拉枪。这有一个好处：新玩家可以快速上手。但是久而久之你就对它形成依赖了。

和平精英，以及穿越火线手游，默认都把辅助瞄准打开了。穿越火线手游更离谱：大狙打开瞄准镜，准星直接跟着敌人动。这时你的脑回路是这样的：碰见敌人了 -> 按开枪键的同时稍微转一下视角 -> 敌人挂了。

这样你连拉枪的意识都没了。就算有意识，在近身作战的时候，你拉的同时算法也在拉。你和算法在互相争夺枪的控制权：你手指在触摸屏上移动的距离，和准星实际移动的距离不是线性关系，导致你永远掌握不了那个度。

——这是笔者从哔哩哔哩的一位名叫时少权的 up 主那里听来的。他对自己比较狠，不用手指头转视角，用陀螺仪。他这个人在打游戏的时候是站起来的，用身子上下左右晃，带动手机传感器，再带动游戏里的人物视角转动，跟打 VR 游戏一样。他还比较骇俗，以前是做游戏直播和出视频教程，现在好像做动画去了。

找掩体、卡视角

和平精英分第三人称模式和第一人称模式。我们一般玩的是第三人称模式，就是可以看到你这个人的身子。第一人称只能看到你真正应该看到的东西。

在第三人称模式下，信息是不对等的：你可以在一个墙角卡视角，看到侧面过来的人，但是对方看不到你。

这在团队竞技模式下非常有用。团竞地图本来就小，4v4。所以团队竞技又叫卡点竞技。

野外作战的时候，拿反斜坡、石头、树或者载具作掩体是生存的基本思路。还有烟雾弹不是摆设，除了救人，跑圈转移的时候也可以用，这是在你位置已经暴露或者有很大风险暴露的情况下。

团队竞技要有点团队意识

一般来说，随机匹配的队友都不会开麦，就算开麦也没人指挥、报点，各打各的。

这时你可以有点团队意识：跟在一个队友后面捡漏。

金手指、TAS 与 RTA

金手指是内存条上的金黄色导电触片。代指修改游戏内存，实现作弊。

TAS 原指 Tool-Assisted Speedrun（工具辅助竞速）。是在不修改游戏内存（但可以监控，以及利用游戏 Bug）的前提下，人理论上可以达到的最好成绩。

TAS 的原理是：

  • 录制输入：记录下游戏每一帧的外部输入（按键/鼠标）。比如在第 300 帧，按下了【右】【A】这两个键；第 301 帧，按下了【右】【A】【B】这三个键……
  • 逐帧操作：想办法让游戏程序一帧一帧地执行，我们一帧一帧地输入
  • 存档/读档：如果我们对第 300 帧的输入不满意，可以重新输入。每个帧都可以存一个档，包括在这一帧之前的所有输入记录。
  • 通过一系列的输入，最终让游戏加载出通关动画。这包括利用游戏 Bug，让游戏内存溢出。比如《超级马里奥世界》。

最后把游戏每一帧的外部输入记录成一个文件，这被称作录像文件（不是视频）。还包括录像作者，重录（存/读档）次数等。

对于不追求通关速度，而是表演性的 TAS 来说，S 指 Superplay。比如超级马里奥一代[最多金币]世界记录。

RTA（Real Time Attack）是真实时间通关，是不借助任何工具的实时操作，是人实际上的最好成绩。

混音

  1. 有问题/有死线
  2. 有扳机
  3. 把软件装上

重要性 1 > 2 > 3

混音就是把好几堆数字合并，并转换为另一堆数字，使转换后的数字客观听起来好听。

声源 -> 电脑 -> 耳朵

声源 相关名词
人唱歌 声乐
人演奏 器乐、乐器、演奏
机器唱歌 歌声合成技术、声库、VOCALOID
机器演奏 软件音源、编曲
  • 人唱歌/演奏 -> 通过麦克风录制（电磁学） -> 通过声卡把模拟信号转换为数字信号 -> 通过声卡 ASIO 驱动给到电脑操作系统 -> 操作系统给到 DAW ->
  • 声库/软音源 -> 给到 DAW ->
    • 存到内存里 -> 在 DAW 里用 VST 插件等进行处理（这一步才叫混音） ->
      • 渲染（导出、编码）文件
        • 音频文件 -> 解码器 -> 播放器 ->
        • 内存里的音频 -> DAW 实时回放 ->
          • 音频系统（ASIO、WASAPI 等）-> 输出 -> 耳朵

其中 ASIO 驱动是音频流输入输出程序， DAW（Digital Audio Workstation）是数字音频工作站，即编曲软件，也叫 VST 宿主机。VST 意思是 Vitrual Studio Technology（虚拟工作室技术），免费的 DAW 有 Reaper 等。

混音是一项需要非常强理性和感性的工作。如果你能做好混音，说明你心理素质非常强大。笔者现在做不到。

混音的关键是做平衡。因此得有一些标准

名词

驻波：两波叠加时出现的稳定加强/减弱点。

共振：人荡秋千，你推的频率要和他荡的频率相同，他才荡得高。你对一首歌有共鸣，说明这首歌对上你的电波了。

泛音列：

  1. 复合音 = 基音 + 泛音列
  2. 各列泛音为基音的整数倍
  3. 音色区别在于各列音的振幅（能量）不同
  4. C：CEG 和谐是因为 C 的泛音中“占比最多”为 C>G>E C5：CG 则被称作 C 强力五和弦

音频文件属性

把音频文件看成是一条离散采样的波形

采样率

采样的时间精密度，水平方向的。一秒钟之内采样多少个。

人耳听力范围约 \(20 \sim 20000 \mathrm{Hz}\)。按奈奎斯特采样定理，采样率要大于最高频率两倍，才可较准确还原。因此最常用（CD）的采样率为 \(44100 \mathrm{Hz}\)

或者记：记录一次振动，最少要采两个点

比特率

采样的空间精密度（深度），竖直方向的。用多少个比特切割“振幅”。

比如 \(16 \mathrm {bit}\) 的位深，对于每一个采样，有 \(2^{16}=65536\) 种度量“振幅”的刻度。

\(1 \mathrm{bit}\) 对应 \(6\mathrm{dB}\) 动态范围（？）。

波形统计信息

用 RX/AU 等软件可以查看音频文件的频谱图。横轴是时间，纵轴是频率，颜色亮度代表电平（振幅/能量）

用 RX 的 Waveform Statistic 查看：

Max. RMS level db

四大动态效果器

改变动态范围的。

  • 压缩器：把超出门槛的波形压小
  • 限制器：大号压缩器，把超出门槛的波形限制到门槛以下
  • 扩展器：把门槛以下的波形压小
  • 门限器（噪音门）：把门槛以下的波形去了

社保是什么

社保是社会保险。再往前问一步，保险是什么？

你交钱给一个组织，并和他们签订契约。当你出事且符合出事的条件时，这个组织会根据契约的条文返还你对应比例的钱。

保险类型 组织 契约 出事的条件
商业保险 保险公司 保险合同、法律、法规 由契约规定
社会保险 国家 法律、法规 由契约规定

保险是一种投资，当然可以选择不交——如果你不担心：

  • 老了，没钱用
  • 看病，没钱用
  • 工作受伤，没钱用
  • 工作没了，没钱用
  • 生孩子，没钱用

这五大问题的话。它们分别对应社保的：

  • 基本养老保险
  • 基本医疗保险
  • 工伤保险
  • 失业保险
  • 生育保险

城镇职工社保和城乡居民社保

它们都属于社会保险。

  • 城镇职工社保，包括上面的五险。
  • 城乡居民社保，只包括医疗保险和养老保险。
社会保险类型 谁交 交的前提 交的时间
职工社保 职工和雇佣者分担 职工与雇佣者签订劳动合同 按月
