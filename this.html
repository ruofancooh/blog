MMPI

统计学家 George Box 说：所有模型都是错的，有些是有用的。这里研究的是对于明尼苏达多相人格测验（Minnesota Multiphasic Personality Inventory）的一系列判断题，回答是或否对最终结果的影响。依据是知乎网友 aweeee分享的 xlsx 文件。

  1. 做一道判断题，回答是或否

  2. 答完一道题之后，对某一个或者某几个量表加 1 分或不加分

  3. 答完所有题之后，会得到十四个分，包括四个效度量表（Q L F K）和十个临床量表（Hs D Hy Pd Mf Pa Pt Sc Ma Si）。这十四个分称作原始分。如果你做的是 566 道题的，额外会有五个附加量表（MAS Dy Do Re Cn）。笔者做的都是两百道题的，只有这十四个分。

  4. 特别地，对这五个临床量表加上一定比例的 K 分，加完之后的称为 K 校正分

    • Hs += 0.5K
    • Pd += 0.4K
    • Pt += 1.0K
    • Sc += 1.0K
    • Ma += 0.2K

  5. 现在我们有十四个分。对除 Q 之外的十三个分分别计算 T 分：

    \[ T = 50 + \frac{10 \times (你的得分 - 常模均值)}{常模标准差} \]

    其中，【常模均值】和【常模标准差】由你的国籍、性别和年龄范围选择。

    【常模均值】是一群正常人（健康人）做的平均分。用你的得分与其相减，刻画的是你与正常人的偏离程度。

    【常模标准差】刻画的是这一群正常人得分的离散程度。用其作分母，可以缩小误差。

四个效度量表，测你做的结果是否有效：

Q（Question）【疑问】：你没有回答的题目数

在实际测验中，与“性”有关的几个题目可以选择跳过回答。

L（Lie）【说谎】：你往圣人方向靠的程度

例如

做游戏的时候，我只愿赢而不愿输

偶尔我会想到一些坏得说不出口的事

有时我也讲假话

等的这一些问题，回答【否】会加分

F（Validity）【诈病】：你装病的程度，或者判断力丧失的程度

例如

有时我觉得有鬼神附在我身上

我能在我周围看到其他人所看不到的东西、动物和人

等的这一些问题，回答【是】会加分

K（Correction）【校正】：你对自己的私事隐瞒的程度、防御的程度

例如

当我正在做一件重要事情的时候，如果有人向我请教或打扰我，我会不耐烦的

我觉得许多人喜欢夸大自己的不幸，以便得到别人的同情和帮助

我不在乎别人对我有什么看法

回答【否】会加分

我几乎没有和家里人吵过嘴

回答【是】会加分

其他

金手指、TAS 与 RTA

金手指是内存条上的金黄色导电触片。代指修改游戏内存，实现作弊。

TAS 原指 Tool-Assisted Speedrun（工具辅助竞速）。是在不修改游戏内存（但可以监控，以及利用游戏 Bug）的前提下，人理论上可以达到的最好成绩。

TAS 的原理是：

  • 录制输入：记录下游戏每一帧的外部输入（按键/鼠标）。比如在第 300 帧，按下了【右】【A】这两个键；第 301 帧，按下了【右】【A】【B】这三个键……
  • 逐帧操作：想办法让游戏程序一帧一帧地执行，我们一帧一帧地输入
  • 存档/读档：如果我们对第 300 帧的输入不满意，可以重新输入。每个帧都可以存一个档，包括在这一帧之前的所有输入记录。
  • 通过一系列的输入，最终让游戏加载出通关动画。这包括利用游戏 Bug，让游戏内存溢出。比如《超级马里奥世界》。

最后把游戏每一帧的外部输入记录成一个文件，这被称作录像文件（不是视频）。还包括录像作者，重录（存/读档）次数等。

对于不追求通关速度，而是表演性的 TAS 来说，S 指 Superplay。比如超级马里奥一代[最多金币]世界记录。

RTA（Real Time Attack）是真实时间通关，是不借助任何工具的实时操作，是人实际上的最好成绩。

混音

混音就是把好几堆数字合并，并转换为另一堆数字，使转换后的数字客观听起来好听。当然，只说说当然是不难的。

声源 -> 电脑 -> 耳朵

声源 相关名词
人唱歌 声乐
人演奏 器乐、乐器、演奏
机器唱歌 歌声合成技术、声库、VOCALOID
机器演奏 软件音源、编曲
  • 人唱歌/演奏 -> 通过麦克风录制（电磁学） -> 通过声卡把模拟信号转换为数字信号 -> 通过声卡 ASIO 驱动给到电脑操作系统 -> 操作系统给到 DAW ->
  • 声库/软音源 -> 给到 DAW ->
    • 存到内存里 -> 在 DAW 里用 VST 插件等进行处理（这一步才叫混音） ->
      • 渲染（导出、编码）文件
        • 音频文件 -> 解码器 -> 播放器 ->
        • 内存里的音频 -> DAW 实时回放 ->
          • 音频系统（ASIO、WASAPI 等）-> 输出 -> 耳朵

其中 ASIO 驱动是音频流输入输出程序， DAW（Digital Audio Workstation）是数字音频工作站，即编曲软件，也叫 VST 宿主机。VST 意思是 Vitrual Studio Technology（虚拟工作室技术），免费的 DAW 有 Reaper 等。

采样率

采样的时间精密度，水平方向的。一秒钟之内采样多少个。

人耳听力范围约 \(20 \sim 20000 \mathrm{Hz}\)。按奈奎斯特采样定理，采样率要大于最高频率两倍，才可较准确还原。因此最常用（CD）的采样率为 \(44100 \mathrm{Hz}\)

或者记：记录一次振动，最少要采两个点

比特率

采样的空间精密度（深度），竖直方向的。用多少个比特切割“振幅”。

比如 \(16 \mathrm {bit}\) 的位深，对于每一个采样，有 \(2^{16}=65536\) 种度量“振幅”的刻度。

《和平精英》战略战术

《和平精英》是一款 FPS（First-person shooting，第一人称射击）的手机端游戏，由腾讯光子工作室群研发。沿袭了端游《绝地求生》（PUBG）的玩法。

不要开辅助瞄准

辅助瞄准就是算法帮你拉枪。这有一个好处：新玩家可以快速上手。但是久而久之你就对它形成依赖了。和平精英，以及穿越火线手游，默认都把辅助瞄准打开了。穿越火线手游更离谱：大狙打开瞄准镜，准星直接跟着敌人动。这时你的脑回路是这样的：碰见敌人了 -> 按开枪键的同时稍微转一下视角 -> 敌人挂了。这样你连拉枪的意识都没了。就算有意识，在近身作战的时候，你拉的同时算法也在拉。你和算法在互相争夺枪的控制权：你手指在触摸屏上移动的距离，和准星实际移动的距离不是线性关系，导致你永远掌握不了那个度。这是笔者从哔哩哔哩的一位名叫时少权的 up 主那里听来的。他对自己比较狠，不用手指头转视角，用陀螺仪。他这个人在打游戏的时候是站起来的，用身子上下左右晃，带动手机传感器，再带动游戏里的人物视角转动，跟打 VR 游戏一样。他还比较骇俗，以前是做游戏直播和出视频教程，现在好像做动画去了。

戴耳机

这也是最基本的一点，最基本的听声辨位，只要你戴耳机都可以听出来。除了电竞模式之外，在小地图上都会显示脚步来自哪个方位。但你不可能永远盯着小地图看。高阶的听声辨位包括：你在二楼，敌人在一楼、二楼、三楼的脚步声音是不同的。

找掩体、卡视角

和平精英分第三人称模式和第一人称模式。我们一般玩的是第三人称模式，就是可以看到你这个人的身子。第一人称只能看到你真正应该看到的东西。在第三人称模式下，信息是不对等的：你可以在一个墙角卡视角，看到侧面过来的人，但是对方看不到你。这在团队竞技模式下非常有用。团竞地图本来就小，4v4。所以团队竞技又叫卡点竞技。野外作战的时候，拿反斜坡、石头、树或者载具作掩体是生存的基本思路。还有烟雾弹不是摆设，除了救人，跑圈转移的时候也可以用，这是在你位置已经暴露或者有很大风险暴露的情况下。

团队竞技要有点团队意识

一般来说，随机匹配的队友都不会开麦，就算开麦也没人指挥、报点，各打各的。这时你可以有点团队意识：跟在一个队友后面捡漏。

进圈要紧

现在到最后几个圈了，你还在毒里，向安全区跑。这时你看到五十米开外有个人也在跑圈，他没看见你，请问你打不打？如果打起来了，活下来的一方一是浪费时间，二负伤，三是暴露位置给圈里的人了。你们都活不了。还有一个套路：要么做最早进圈的，要么做最晚进圈的。度假岛地图小、药多、刷圈快，进决赛圈之前，你可以一直在毒里打药。跑得快不一定赢，不跌跟头才是成功。

先解决最重要的

一般来说，你在瞄远处的人的时候，身边来人了，你会先注意身边的人。特殊情况：你和敌人在掩体的两头对峙，同时侧面的房顶有人在瞄你。你会和房顶的人对枪。

打一枪换一个地方

这句话不是没有道理的。特别是在团竞地图，你把一个敌人解决了，除了他马上就复活以外，其他敌人也会知道你的方位。如果你卡点能力特别强，一直卡同一个点——敌人也不会给你送了，或者打的是占点竞技/躲猫猫，那就没必要。
